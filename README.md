# SENG3030 IoT Project

## Setup

1. Clone this repo
2. Open in VS Code with PlatformIO
3. Copy `src/config.example.h` to `src/config.h`
4. Fill in your WiFi and MQTT credentials in `config.h`
5. Build and upload

## Hardware

- M5StickC

## Current Features (Person 1)

- Sends sensor data every 1 second via MQTT
- Side button (BtnB) toggles sending on/off
- Screen: Blue = off, Green = on
